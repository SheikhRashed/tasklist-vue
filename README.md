## Installation

Install my-project with yarn

```bash
  git clone https://github.com/SheikhRashed/tasklist-vue.git
  
  cd tasklist-vue
  
  code .

  yarn 

  yarn dev 
```
Install my-project with npm

```bash
  git clone https://github.com/SheikhRashed/tasklist-vue.git
  
  cd tasklist-vue
  
  code .

  npm install 

  npm run dev 
```
